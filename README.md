# 🚀 local_timehub

> embedded key-value store for local apps

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g local_timehub
local_timehub init my-project
```

## 🛠️ Stack

Built with swift-png + fastbook
